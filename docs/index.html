TypeScriptへのトランスファイル方法
  • tscコマンドを実行してjsファイルをtsにトランスファイルする。 以下のコマンドでトランスファイル。
    $ ./node_modules/.bin/tsc -p .
    その後、index.jsを実行すると以下のような結果を得られる
    $ node index.js
